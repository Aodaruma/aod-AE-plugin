# singular-value-decompose ( AOD_SingularValueDecompose )

Performs 2D singular value decomposition and low-rank approximation on RGBA channels.

This is the After Effects plugin **AOD_SingularValueDecompose**, which provides the **SingularValueDecompose.aex** plugin file for Adobe After Effects.
