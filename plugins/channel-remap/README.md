# channel-remap ( AOD_ChannelRemap )

Remaps RGBA channels from source channels, constants, and layers with selectable color spaces.

This is the After Effects plugin **AOD_ChannelRemap**, which provides the **ChannelRemap.aex** plugin file for Adobe After Effects.
