# eyedropper-mask ( AOD_EyedropperMask )

Adjusts opacity of contiguous regions around eyedropper points with per-point controls and selectable premultiplied or straight workflows.

This is the After Effects plugin **AOD_EyedropperMask**, which provides the **EyedropperMask.aex** plugin file for Adobe After Effects.
